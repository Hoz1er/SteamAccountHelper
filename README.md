# SteamAccountHelper
Steam账号快捷选择，自动登录

## Requires
  * Windows
  * .Net Framework 4.5+

## Usage
打开本程序后直接显示出全部可自动登录的账号
* 双击"使用其他账号登录"：  
不自动登录，手动填写账号密码登录。  
如果登录时将该账号设为可自动登录，则下次打开本程序，列表中能够看到该账号
* 双击已有账号：  
以该账号登录Steam  
如果没有看到期望的账号，请先以常规方式启动Steam，以该账号登录，并确保将该账号设置为可自动登录。
